A dedicated component of the **ftw-pki** suite focusing on signing certificates using an Intermediate Certificate Authority.

## 🛠 Features

* **Intermediate Signing Logic:** Specialized tools for issuing end-entity certificates (Server/User) from an Intermediate CA.
* **Security Profiles:** Integrated validation and policy enforcement for intermediate-level issuing.
* **Compatibility:** Seamless interaction with CSRs generated by the `ftw-pki-intermediate-creator`.

## 📖 Documentation & Usage

This tool handles the signing process within the chain of trust.

* **CLI:** Use the `ftwpkiintermedsigner` utility (once defined in pyproject.toml).
* **Technical Manual:** See `doc/source/devel/` for the module documentation and integrated doctests.
